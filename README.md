# GH-900

GH-900 수업 데모용 리포지토리

## 📋 리포지토리 개요

이 리포지토리는 **GH-900 수업**을 위한 데모 프로젝트입니다. GitHub와 Git의 기본 기능들을 학습하고 실습하기 위해 만들어졌습니다.

## 📁 리포지토리 구조

```
GH-900/
├── README.md          # 프로젝트 설명 문서 (이 파일)
├── LICENSE            # MIT 라이선스 파일
└── .github/
    └── FUNDING.yml    # GitHub 후원 설정 파일
```

## 🎯 목적

- GitHub 리포지토리 생성 및 관리 학습
- Git 기본 명령어 실습
- 마크다운(Markdown) 문서 작성 연습
- 오픈소스 프로젝트 구조 이해
- 라이선스 및 문서화 모범 사례 학습
